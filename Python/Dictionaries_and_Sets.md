Dictionaries
------------

* Key-value pair to store properties
* Syntax:
```python
Student_details = {
    "name": "John",
    "age": 15,
    "class": "X",
    "Gender": "male"
}

print(Student_details["name"])
```

```python
Students_info = [
    {
        "name": "John",
        "age": 15,
        "class": "X",
        "Gender": "male"
    },
    {
        "name": "Tim",
        "age": 14,
        "class": "IX",
        "Gender": "male"
    },
    {
        "name": "Kay",
        "age": 16,
        "class": "XI",
        "Gender": "female"
    }
]

print(Students_info[2]["age"])
```
